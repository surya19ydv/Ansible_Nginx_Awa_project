Ansible + AWS Automation Project

Nginx Deployment on Multiple Servers using Ansible

Project Overview

This project demonstrates how to automate web server deployment using Ansible on AWS EC2 instances.

  • 1 Control Node (Ansible Installed)
  • 2 Managed Nodes (Web Servers)
  • Nginx Installation
  • Website Deployment

Architecture

Control Node
(Ansible)
Server 1
(Nginx)
Server 2
(Nginx)

Inventory File

[web]
server1 ansible_host=IP1
server2 ansible_host=IP2

Ansible Playbook

---
- name: Nginx Install and Deployment
  hosts: webserver
  become: yes

  tasks:
    - name: Install Nginx
      apt:
        name: nginx
        state: present
        update_cache: yes

    - name: Start and Enable Nginx
      service:
        name: nginx
        state: started
        enabled: yes

    - name: Deploy Website
      copy:
        src: index.html
        dest: /var/www/html/index.html
        force: yes

Website Code

<h1>Welcome to My Automated Server</h1>
<p>Deployed using Ansible</p>

Execution Command

ansible-playbook -i inventory playbook.yml
